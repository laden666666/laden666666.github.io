picker

高仿IOS的PickerView的JavaScript插件，3D形式的滚轮选择器，同时支持最多三级联动的选择效果，支持pc端和移动端两种平台浏览器。

简单的选择器使用

无 new picker({ cols: ['java','c#','JavaScript','php','Python'], onOkClick: function (values) { document.querySelector('#example1').textContent = (values[0]); }, })
